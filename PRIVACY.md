# Privacy Policy

**Effective date:** July 11, 2026

Stacey's Chrome Presentation Tool does not transmit, sell, or share personal information or browsing data with the developer or any third party. It handles limited data locally only to provide its user-requested features.

The extension stores only the most recently used window dimensions in Chrome's local extension storage on the user's device. This setting is used solely to restore the user's last selected size. It is not sent anywhere.

When the user selects **Capture Visible Tab**, the extension temporarily accesses the currently visible website content to create a PNG and downloads that image directly to the user's device. The image is not uploaded, transmitted, or retained by the extension.

The extension makes no network requests and includes no analytics, advertising, tracking, or third-party services.

Stacey's Chrome Presentation Tool's use of user data complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.

For privacy questions, contact the publisher through the Chrome Web Store listing.
